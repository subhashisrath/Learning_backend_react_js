# The Master Learning Path: From Fundamentals to Job-Ready

Building 10 small things is better than building 1 big thing poorly. Each project in this list introduces **one major new concept**.

## Project 1: Expense Tracker (The Basics)

- **Goal**: Master HTTP, Express, and Basic SQL.
- **Key Concepts**: REST APIs, MySQL connection, Environment Variables, CRUD.

## Project 2: Task Architect (Authentication & Relations)

- **Goal**: Secure an application and link data.
- **Key Concepts**: JWT (Authentication), Hashing Passwords (Bcrypt), One-to-Many Relationships (SQL Joins).

## Project 3: File Locker (File Streams & Storage)

- **Goal**: Handling non-text data.
- **Key Concepts**: Multer (File uploads), File System (Node `fs`), Static file serving.

## Project 4: Real-time Chat (WebSockets)

- **Goal**: Move beyond Request-Response.
- **Key Concepts**: Socket.io, Bi-directional communication, Event handling.

## Project 5: Mini E-Commerce Engine (The Boss Level)

- **Goal**: Complex State & Payments.
- **Key Concepts**: Many-to-Many relationships, Payment Gateway integration (Stripe/Razorpay), Transaction management in SQL.

---

### Why this order?

We don't do E-commerce first because you'll get stuck on "How do I secure the checkout?" before you even know "How do I save a product?". We build the habits first.
